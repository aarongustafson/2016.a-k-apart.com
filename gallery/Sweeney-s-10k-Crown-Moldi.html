Sweeney's 10k Crown Molding Calc-O-Matic

by Michael Sweeney

This is a simple but hopefully useful crown-molding calculator that records the spring (crown) and corner angles you enter, calculates the resulting miter and bevel angles, and provides a diagram to help you visualize how to orient the molding on your saw. The Calc-O-Matic is static HTML with JavaScript and CSS tranforms providing the essential functionality. It adds the entered angles to local browser storage so you don't lose the key data points until you're done with your cuts.

View the source of this project on GitHub: https://github.com/codenamesweeney/crown-molding-calc-o-matic

Digital Clock

by Wouter Lemoine

This is a simple digital clock, constructed and animated with CSS. Javascript is only used to get the time. Also, a screenreader is able to read the hours and minutes, thus speaking out the time.

View the source of this project on GitHub: https://github.com/wouterlemoine/10k-numberwatch

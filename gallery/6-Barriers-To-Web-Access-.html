6 Barriers To Web Access For The Visually Impaired

by Chris Lona

This presentation shows 6 barriers the visually impaired face when accessing the web. Also included is a link to a page with a video of how the page is experienced with a screen reader. It's important to remember that web access for the 3 other main disability types (auditory, physical, cognitive) is a series of separate solutions. Except in the case of the cognitively challenged where there are few concrete recommendations. Our goal is to provide a new perspective about web and digital access.

View the source of this project on GitHub: https://github.com/clona/6-Barriers-To-Web-Access
